Track the project

These lists are read only.

  • announce@dartlang.org
    Low traffic announcements of new releases, breaking changes, and other important news. Recommended!
  • commits@dartlang.org
    High-traffic list of all commits to the main Dart repository.
  • bugs@dartlang.org
    High-traffic list of all updates and changes from across the main Dart issue tracker.
  • reviews@dartlang.org
    High-traffic list of all comments and code reviews from the main Dart repo.
  • news.dartlang.org
    Subscribe to the official news feed for the Dart project.
